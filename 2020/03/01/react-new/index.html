react核心概念梳理

发表于 2020-03-01 | 分类于 前端

学习react(基于15.6)过程中理解核心概念还是很重要的，便于我们抓住主干线索

  • element(虚拟DOM)

  • instance
    平时我们写的class/函数组件的实例，包含生命周期等
    state/props变化后，触发更新逻辑

  • 虚拟DOM管理类
    根据虚拟DOMtype创建的管理类，主要用来管理和维护虚拟DOM。内部拥有instance，用来获取新rendered元素、触发生命周期

    • ReactCompositeComponent
      用来管理element.type是函数的虚拟DOM元素
    • ReactDOMComponent
      用来管理element.type是string(div,span)的虚拟DOM元素
